## Quick Reference

| Command | Description | Permission |
|---------|-------------|------------|
| `/broadcast <message>` | Broadcast a message to all servers | `fairychat.command.broadcast` |
| `/pm <username> <message>` | Send a private message | None |
| `/r <message>` | Reply to a private message | None |
| `/ignore <username>` | Toggle ignoring a player | None |
| `/clearchat [player]` | Clear the chat | `fairychat.command.chatclear` |
| `/fairychat` | Show plugin information | None |
| `/fairychat reload` | Reload the configuration | `fairychat.command.fairychat.reload` |
