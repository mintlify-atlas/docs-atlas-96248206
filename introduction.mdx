# Welcome to FairyChat

FairyChat is the future of chat management for your server, providing a range of features from customizable chat formats to multi-server message broadcasting. The perfect solution to bring a seamless chatting experience to your server users.

## Key Features

<CardGroup cols={2}>
  <Card title="Customizable Chat Format" icon="palette">
    Personalize the chat format as per your needs with per-group chat formats to differentiate chats based on user groups for better management.
  </Card>
  
  <Card title="Private Messaging" icon="envelope">
    Send private messages to other players with ease. Supports tab completion across multiple servers.
  </Card>
  
  <Card title="Player Mentions" icon="at">
    Mention other players in chat using the `@` symbol. Get notified with a sound when you are mentioned.
  </Card>
  
  <Card title="Multi-Server Support" icon="server">
    Optional multi-server message sending with Redis. Send messages, broadcasts, and private messages across multiple servers seamlessly.
  </Card>
  
  <Card title="MiniMessage & MiniPlaceholders" icon="code">
    All formats are compatible with MiniMessage & MiniPlaceholders, making the customization process easier than ever.
  </Card>
  
  <Card title="Legacy Support" icon="clock-rotate-left">
    Legacy color codes are supported, so you don't have to worry about your old chat formats. PlaceholderAPI is also supported.
  </Card>
</CardGroup>

## Additional Features

<CardGroup cols={3}>
  <Card title="Chat Cooldown" icon="hourglass">
    Prevent spamming by adding a cooldown to chat messages with a configurable threshold.
  </Card>
  
  <Card title="Similarity Check" icon="scale-balanced">
    Prevent players from sending similar messages in a short period of time.
  </Card>
  
  <Card title="Player Ignore" icon="ban">
    Ignore players you don't want to see messages from.
  </Card>
  
  <Card title="Chat Clear" icon="broom">
    Clear the chat with a single command in multiple servers at once.
  </Card>
  
  <Card title="Reloadable Config" icon="rotate">
    All configuration changes can be reloaded without restarting the server.
  </Card>
  
  <Card title="Fully Paper Compatible" icon="feather">
    FairyChat is fully compatible with Paper and optimized to provide the best performance.
  </Card>
</CardGroup>
